# 个人自用魔兽争霸 3 自定义快捷键

- 魔兽争霸3冰封王座版本的快捷键要在游戏内打开自定义快捷键，然后把 [这个文件](<Warcraft III Frozen Throne/CustomKeys.txt>) 放入根目录下

- 魔兽争霸3重制版的快捷键也要在游戏内开启自定义快捷键，把 [这个文件](<Warcraft III Reforged/CustomKeys.txt>) 放在 `C:\Users\Admin\Documents\Warcraft III\CustomKeyBindings` 文件夹下

目前冰封王座版本的快捷键只修改了人族，重制版的全修改了
